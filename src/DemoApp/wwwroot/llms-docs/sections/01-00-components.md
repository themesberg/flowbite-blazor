## Available Components

Flowbite Blazor provides the following set of UI components:
- Alert
- Avatar
- Badge
- Breadcrumb
- Button
- Card
- Carousel
- Chatbot
- Drawer
- Dropdown
- EmptyState
- Form Components
   - TextInput
   - TextArea
   - Select
   - Checkbox
   - Radio
   - FileInput
   - ToggleSwitch
   - RangeSlider
- Modal
- Icons
- Navbar
- Pagination
- QuickGrid
- Sidebar
- Skeleton
- Spinner
- Table
- Tabs
- Timeline
- Toast
- Toolbar
- Tooltip
- Typography (Heading, Paragraph, Span)
