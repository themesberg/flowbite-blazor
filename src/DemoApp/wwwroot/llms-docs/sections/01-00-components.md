## Available Components

Flowbite Blazor provides the following set of UI components:
- Alert
- Avatar
- Badge
- Breadcrumb
- Button
- Card
- Drawer
- Dropdown
- Form Components
   - TextInput
   - TextArea
   - Select
   - Checkbox
   - Radio
   - FileInput
   - ToggeSwitch
   - RangeSlider
- Modal
- Icons
- Navbar
- QuizGrid
- Sidebar
- Spinner
- Tooltip
